# **App Name**: LearnAI Roadmap

## Core Features:

- Learning Goal Input: Allows users to input their learning goals, skill level, and daily time commitment.
- AI Roadmap Generation: Uses an AI tool with a configurable API key to generate a structured learning roadmap with estimated times based on the user's input.
- Roadmap Display: Displays the AI-generated roadmap in a split-screen layout, with roadmap steps on the left and detailed information on the right.
- Interactive Step Details: Allows users to click on roadmap steps to view detailed information, including what they will learn, first steps, and estimated mastery time.
- User Authentication: Requires users to log in or register before they can save a roadmap. Redirects unauthenticated users to the login/registration page.
- Save Roadmap: Allows authenticated users to save generated roadmaps to their user accounts.
- View Saved Roadmaps: Allows users to view a list of their saved roadmaps.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) to represent a clear path and trust in the learning journey.
- Background color: Light blue (#E3F2FD) to provide a calming and uncluttered backdrop for the roadmap content.
- Accent color: Teal (#26A69A) for highlighting interactive elements and important information, offering a contrasting yet harmonious feel.
- Body font: 'PT Sans', a humanist sans-serif providing readability for roadmap details and descriptions.
- Headline font: 'Space Grotesk', a contemporary sans-serif, is used for headers.
- Use clean and simple icons to represent roadmap steps and categories.
- Maintain a split-screen layout with a clear visual hierarchy.
- Employ subtle transition animations between steps for enhanced interactivity.
